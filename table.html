Html Vs Html5 :

Basis Html Html-5
Mobile Friendliness Html is less mobile friendly Html is more mobile friendly
Storage option For storage browser cache can be used as temporary storage For storage Application cache, web SQL database, and web storage can be used in HTML5
Drag and Drop Feature It doesn't support drag and drop effects. It does support drag and drop effects.
Javascript Support Doesn't support Javascript to run in browser Support javascript to run in background with the help of js-Web
Media support Doesn't support audio & video without use of flash player Support Audio & video with use of audio & video tag
Vector Graphics Html uses vector graphics with the help of various technologies such as VML,FLASH,etc. Vector Graphics is an important part of HTML5 as we use SVG and canvas in it.
